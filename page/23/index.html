理解python中的列表推导

发表于 2014-03-04 | 编程实践 ， python

2014-03-04 19:20

列表推导（List comprehension）的作用是为了更方便地生成列表（list）。

比如，一个list变量的元素均为数字，如果需要将每个元素的值乘以2并生成另外一个list，下面是一种做法：

#-*-encoding:utf-8-*-

list1 = [1,2,4,5,12]
list2 = []
for item in list1:
    list2.append(item*2)
print list2
