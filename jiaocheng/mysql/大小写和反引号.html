大小写和反引号

大小写和反引号

大小写

默认配置下，关键词是不区分大小写的。例如 CREATE TABLE、create table create Table

函数也不区分大小写。

反引号

反引号，可以用在数据库名、表名、字段名、索引名上。

反引号必须用的场合：表名、字段名用了 MySQL 保留字。其他情况下，可用可不用。

注意，切换数据库时不能用反引号：

use `bank`;  -- 会报错
