MySQL管理工具

MySQL管理工具

mysql 命令

mysql 命令是 MySQL 自带的命令行客户端工具。

mycli 命令

mycli 是用 python 编写的一个命令行客户端，支持自动补全和语法高亮。

地址：https://github.com/dbcli/mycli

MySQL Workbench

MySQL 官方的可视化管理工具。

Navicat

一个优秀的可视化管理工具。

DataGrip

JetBrains 出品的可视化 IDE。

SQL 格式化工具

一些可视化工具是自带格式化功能的。

这是一个网页版工具：http://tool.oschina.net/codeformat/sql 。
