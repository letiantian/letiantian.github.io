DDL、DML、DCL等的含义

DDL、DML、DCL等的含义

DDL

Data Definition Language，数据定义语言。

主要是 CREATE、ALTER、DROP 等。

DML

Data Manipulation Language，数据操纵语言。

是 SELECT、INSERT、UPDATE、DELETE。

DCL

Data Control Language，数据控制语言。

主要是 GRANT、REVOKE。
