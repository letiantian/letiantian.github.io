NULL的判等

NULL 的判等，不能用 =，而应该用 is 。判不等，不能用!=，而应该用is not。

示例 执行结果
select null is null; 1
select null = null; null
select 1 = 1; 1
select 1 = 0; 0
select 1 != null; null
select 1 is not null; 1
select null is not null; 0

执行结果中，1 是 true，0是false，null代表未知。

可以这样理解，未知是未知，但两个未知不一定相等。
