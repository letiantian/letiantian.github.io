教程简介

Flask 读音：英 [flɑːsk]，美 [flæsk]

Flask 是一个轻量级的基于 Python 的 Web 框架，支持 Python 2 和 Python 3，简单易用，适合快速开发。

这份文档中的代码使用 Python 3 运行，建议安装最新版本，我使用的是Python 3.6.4。 安装方法，可以自行谷歌或者百度。

建议在 Linux/MacOS 下实践本教程中命令行操作、执行代码。
