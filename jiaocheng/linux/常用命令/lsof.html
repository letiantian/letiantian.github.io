lsof

lsof

查看监听某端口的进行的PID

lsof -i:80

其中 80 是我们要查看的端口。

示例：

$ lsof -i:4000
COMMAND   PID   USER   FD   TYPE             DEVICE SIZE/OFF NODE NAME
node    14297 letian   15u  IPv6 0x9f0618ba48cfd619      0t0  TCP *:terabase (LISTEN)

查看所有进程监听的端口

lsof -i -P
