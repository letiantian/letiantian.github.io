代码高亮之Prism

发表于 2014-04-27 | web

Prism是一个轻量级的可扩展的代码高亮工具。其官方网站是Prism。

配置所需并下载

先到http://prismjs.com/download.html，进行自定义，例如选择压缩版本，theme选择TWILIGHT，选择要高亮的编程语言。以及一些插件，例如Line Numbers。

然后下载prism.js和prism.css文件。

高亮示例

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8" />
<title>Prism</title>
<link rel="stylesheet" href="./prism.css" data-noprefix />
<script src="./prism.js"></script>

</head>
<body>


<pre class="language-markup">
<code>import os
import sys
</code></pre>


<pre class="line-numbers language-css">
<code>h1 {
color:black;
}</code></pre>

</body>
</html>
